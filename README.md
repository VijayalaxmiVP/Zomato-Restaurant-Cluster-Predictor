# Zomato-Restaurant-Cluster-Predictor
This project is a Zomato Restaurant Clustering System developed using Python, Streamlit, and Machine Learning. It uses K-Means Clustering to classify restaurants based on rating, votes, and cost, and predicts the restaurant category through an interactive interface.
Technologies Used
Python
Streamlit
Scikit-learn
K-Means Clustering
PCA
StandardScaler
Features
Predicts restaurant category
Interactive user interface
Machine Learning based clustering
Real-time prediction system
Cluster Categories
Premium Restaurants
Budget Restaurants
Low Rated Restaurants
Popular Mid-Range Restaurants
