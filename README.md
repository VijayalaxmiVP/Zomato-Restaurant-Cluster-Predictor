# Zomato Restaurant Cluster Predictor

This project is a Machine Learning based web application developed using Python and Streamlit. It predicts restaurant clusters based on rating, votes, and approximate cost for two people using the K-Means Clustering algorithm.

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* K-Means Clustering
* PCA
* StandardScaler

## Features

* Predicts restaurant category
* Interactive user interface
* Machine Learning based clustering
* Real-time prediction system

## Cluster Categories

* Premium Restaurants
* Budget Restaurants
* Low Rated Restaurants
* Popular Mid-Range Restaurants
